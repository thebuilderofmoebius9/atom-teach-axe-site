Reference

Glossary

ศัพท์สั้นสำหรับบทเรียน Discord Mirror และ Atom Teach Axe

Backfill

ดึงข้อความย้อนหลังให้กลายเป็นหลักฐานค้นได้ ไม่ใช่รอแต่ข้อความใหม่จาก gateway

FTS5

ระบบ full-text search ใน SQLite เหมาะกับคำถามว่า ‘ข้อความไหนมีคำนี้’

TF-IDF

วิธีให้คะแนนคำที่เด่นในเอกสารหรือช่วงหนึ่งเมื่อเทียบกับทั้งหมด

Judgment

การตีความบริบทหลังเห็นหลักฐาน ไม่ปล่อยให้ keyword ตัดสินแทนทั้งหมด

Presence

บรรยากาศ ความสัมพันธ์ และ context ตอนอยู่ในห้องจริง ซึ่ง mirror แทนได้ไม่ครบ

Mesh

โครงสร้างให้หลายเครื่องหรือหลาย agent ติดต่อกันได้ยืดหยุ่น

Parquet

รูปแบบไฟล์ข้อมูลเชิงตารางสำหรับวิเคราะห์ซ้ำได้

WASM

runtime เบาที่พา logic ไปวิ่งข้ามสภาพแวดล้อมได้

Shell Hygiene

การทำให้ cwd, PATH, env และ output contract สะอาดก่อนให้ agent ทำงาน

Control Room

ชั้น observe/control สำหรับ loop ยาว: status, panel, command, kill switch

STT

Speech-to-text: แปลงเสียงเป็นข้อความก่อนส่งเข้า LLM

Credential Placeholder

เอกสาร public ควรใช้ placeholder แทน secret จริงเสมอ
