Nonce Round-Trip — พิสูจน์การเชื่อมต่อสดที่ปลอมไม่ได้

log หรือ screenshot อ้างว่า bot เชื่อมต่อสำเร็จนั้นปลอมได้เสมอ วิธีพิสูจน์ที่ปลอมไม่ได้คือ nonce round-trip: สร้างค่าสุ่มสดตอนนี้ ส่งเข้าไปในระบบจริง แล้วต้องอ่านค่าเดิมกลับออกมาจากปลายทาง ถ้าค่ากลับมาตรง แปลว่า loop ทั้งเส้นทำงานสดจริง เพราะไม่มีทางเตรียมคำตอบล่วงหน้าได้ ห้องเรียนเพิ่งเห็นตัวอย่างจริง: Gateway reader ที่เขียนเองด้วย raw WebSocket ไม่มี dependency แล้วพิสูจน์ด้วย nonce round-trip

Mission Brief

Mission Brief อ่านย่อ

เปลี่ยนการตรวจรับงานจาก 'ดู log' เป็น 'สั่งระบบพิสูจน์ตัวเองสด' ด้วย nonce round-trip ที่ปลอมไม่ได้

หลักฐานอดีต (log, screenshot, รายงาน) คัดลอกหรือแต่งได้เสมอ แต่ค่าสุ่มที่เพิ่งสร้างแล้ววิ่งครบ loop ปลอมไม่ได้ เพราะไม่มีใครรู้ค่าล่วงหน้า ทีมที่รับมอบงาน integration ควรตั้ง nonce round-trip เป็นเกณฑ์ตรวจรับมาตรฐาน ตัวอย่างจริงจากห้องเรียน: Gateway reader เขียนเองแบบ zero-dependency พิสูจน์ตัวเองด้วยวิธีนี้

ทำไมต้องรู้: ห้องเรียนกำลังไล่ปิดช่อง 'อ้างว่าเสร็จแต่พิสูจน์ไม่ได้' — รอบก่อนปิดเรื่อง pushed ≠ live ด้วย HTTP probe รอบนี้ปิดเรื่อง 'connected จริงไหม' ด้วย nonce round-trip เป็นชุดเครื่องมือตรวจรับที่ใช้คู่กัน

เลนส์ตัดสินใจ: เวลารับรายงานว่า 'ระบบเชื่อมต่อแล้ว' ให้ถามหา liveness proof ที่สั่งได้สด ๆ ไม่ใช่หลักฐานย้อนหลัง ถ้าระบบพิสูจน์สดไม่ได้ ให้ถือว่ายังไม่ผ่านตรวจรับ

จำสั้น ๆ: หลักฐานอดีตปลอมได้ ค่าสุ่มสดที่วิ่งครบ loop ปลอมไม่ได้ — ตรวจรับ integration ด้วย nonce round-trip

Human first

สอนแบบมนุษย์

จำแบบนี้

ถ้ามีคนบอกว่า 'bot ผมต่อ Discord ได้จริงนะ ดู log สิ' เราเชื่อได้แค่ครึ่งเดียว เพราะ log ก็อปวางได้ วิธีที่แน่นกว่าคือให้ระบบพิสูจน์ตัวเองสด ๆ: สร้างเลขสุ่มใหม่ตอนนี้ (nonce) ส่งเข้าระบบจริง แล้วรอให้มันวิ่งกลับออกมาอีกฝั่ง ถ้าเลขที่กลับมาตรงกับที่เพิ่งสร้าง แปลว่าทั้งเส้นทำงานสดจริง เหมือนที่เพื่อนในห้องเรียนเขียน Discord Gateway reader เองด้วย raw WebSocket แล้วพิสูจน์แบบนี้

ทำไมต้องรู้

เพื่อยกระดับจาก 'เชื่อเพราะเห็นหลักฐานอดีต' เป็น 'เชื่อเพราะระบบตอบสด' — หลักฐานแบบหลังปลอมไม่ได้ และใช้ตรวจงานเพื่อน ตรวจ bot ตัวเอง หรือรับมอบงานจากคนอื่นได้อย่างมั่นใจ

ภาพจำ

เหมือนโทรเช็คว่าเพื่อนอยู่บ้านจริงไหม: อย่าดูแค่รูปถ่ายหน้าบ้าน (log เก่า) ให้บอกเพื่อนว่า 'ชูสามนิ้วเดี๋ยวนี้แล้ววิดีโอคอลมา' — ท่าที่เราสั่งสด ๆ ไม่มีทางอัดไว้ล่วงหน้าได้
