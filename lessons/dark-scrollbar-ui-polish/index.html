Mission Brief

Mission Brief อ่านย่อ

ตรวจ visual tokens ให้ครบทั้งพื้นหลัง ขอบ ปุ่ม focus และ scrollbar

free-for-all แปลง observation เล็ก ๆ เป็น durable UI rule ผ่าน memory/readback

เลนส์ตัดสินใจ: ถ้าเว็บเน้นภาพลักษณ์ ต้องมี visual polish checklist ก่อนประกาศ
