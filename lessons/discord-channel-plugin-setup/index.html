Discord Channel Plugin — channel ไม่ใช่ tool ธรรมดา

วันนี้โรงเรียนคุยเรื่องบทความ/โค้ดสำหรับเซ็ต Discord channel plugin และย้ำให้เทียบ source จริง ไม่เชื่อแค่สรุป

Mission Brief

Mission Brief อ่านย่อ

Channel design ต้องเริ่มจาก trust boundary

สรุปบทเรียนจากห้อง: setup Discord plugin ไม่ใช่แค่เปิด bot token แต่ต้องเข้าใจ channel, tool, transport, scope และ access control

ทำไมต้องรู้: ถ้าแยกไม่ออก จะออกแบบ security ผิด เพราะ channel รับ input จากโลกภายนอก ต้องมี gate และ scope ชัด

เลนส์ตัดสินใจ: ถ้าจะทำ plugin จริง ให้ถามก่อนว่า input มาจากใคร เข้ามาทางไหน และถูกจำกัดสิทธิ์อย่างไร

จำสั้น ๆ: channel ต้องมี trust boundary เพราะรับข้อความจากคนภายนอกเข้าหา agent
