Screenshot Proof — ถ้าขอดูภาพ ต้องส่งภาพจริง

ห้องเรียนย้ำว่าเว็บ live ไม่ควรมีแค่ HTTP proof ถ้าผู้ใช้ขอ capture ต้องมี screenshot จริงที่ตรวจว่าไม่ blank และอ่านได้

Mission Brief

Mission Brief อ่านย่อ

สร้าง visual sanity proof ก่อนสรุปว่าเว็บพร้อมอวด

free-for-all มีคำสั่งให้ capture หน้าจอและส่งลิงก์เว็บที่เข้าได้ เป็นมาตรฐานงานเว็บของโรงเรียน

ทำไมต้องรู้: งานเว็บที่ใช้โชว์ครู/ผู้บริหารต้องพิสูจน์ด้วยสายตา ไม่ใช่ header status อย่างเดียว

เลนส์ตัดสินใจ: ถ้าเป้าหมายคือให้คนดูเว็บ ต้องตรวจ output แบบคนดูเว็บ

จำสั้น ๆ: curl พิสูจน์ว่าอยู่, screenshot พิสูจน์ว่าน่าดู
