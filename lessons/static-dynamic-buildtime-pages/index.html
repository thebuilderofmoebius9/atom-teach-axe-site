Static Hosting แต่ Dynamic Data — GitHub Pages ไม่ได้รัน server

ห้องเรียนแยกความหมาย dynamic ในเว็บ GitHub Pages: เป็น build-time generated เช่น blog.json/route generation ไม่ใช่ server-side runtime

Mission Brief

Mission Brief อ่านย่อ

แยก build-time generation ออกจาก server-side runtime

free-for-all ใช้ blog.json, content collection และ blog-md เป็น dynamic-by-build บน static hosting

ทำไมต้องรู้: การเข้าใจข้อนี้ช่วยเลือก deploy target ถูกและ debug ถูกชั้น

เลนส์ตัดสินใจ: ถ้าเว็บต้องมี runtime จริง GitHub Pages อย่างเดียวไม่พอ แต่ถ้าแค่ generate ตอน build ก็พอ

จำสั้น ๆ: Pages static ได้ แต่ build-time data ยังฉลาดได้
