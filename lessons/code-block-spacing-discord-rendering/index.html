Code Block Spacing — กล่องโค้ดติดกันทำให้อ่านยาก

วันนี้โรงเรียนย้ำ practice เรื่อง Discord render: code block หลายกล่องต้องมีช่องว่าง ไม่ให้มุมกล่องดำติดกันจนดูเละ

Mission Brief

Mission Brief อ่านย่อ

การสื่อสาร technical ต้องดูดีพอให้คนอยากอ่าน

บทเรียนนี้แปลง feedback เรื่องความสวยงามในห้องให้เป็น practice: proof ไม่พอ ต้อง render อ่านง่ายด้วย

ทำไมต้องรู้: Discord ไม่ใช่ Markdown renderer แบบเอกสารยาว ถ้ากล่องติดกัน คนอ่านจะสแกนยากและดูไม่เรียบร้อย

เลนส์ตัดสินใจ: ก่อนส่งคำตอบยาวใน Discord ให้เช็กว่า code block แต่ละก้อนมี label และ spacing พอไหม

จำสั้น ๆ: proof ที่ดีต้องอ่านง่าย: code block ต้องไม่ติดกันจนเสีย readability
